#### 💡 专业提示

{% hint style="success" %}
**两者都用!** 在桌面端设置用于详细分析,在移动端设置用于提醒。您的设置会在设备间自动同步。
{% endhint %}

---

#### ❓ 需要帮助?

{% hint style="info" %}
**无法决定?** 大多数交易者使用桌面端进行分析,使用移动端进行监控。如果您现在在电脑前,请从桌面端开始。
{% endhint %}

<summary><strong>购买后看不到指标</strong></summary>

请确保您:

1. 提供了正确的 TradingView 用户名
2. 已刷新浏览器/应用
3. 已检查"仅限邀请脚本 (Invite-only scripts)"部分

仍然有问题? 请联系 <support@infinityalgo.com>

</details>

<details>

<summary><strong>我需要同时设置两个平台吗?</strong></summary>

不需要,您可以选择其中一个或两者都设置。一旦添加到您的账户,您的指标和设置会在设备间自动同步。
