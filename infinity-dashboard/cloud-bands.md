# ☁️ 云形乐队 (Cloud Bands)

自适应波动率包络线（Adaptive Volatility Envelope），随市场条件扩张和收缩，提供动态支撑/阻力位（Support/Resistance）和趋势方向。

---

### 🎯 什么是云形乐队?

云形乐队在价格行为周围创建动态波动率包络线，自动调整以适应市场条件。在波动期间乐队会扩张，在盘整期间会收缩，帮助你识别趋势方向、支撑/阻力位和潜在突破区域。

{% columns %}
{% column width="50%" %}

#### 核心组件

- 🔴 **上乐队 (Upper Cloud)** - 阻力位和超买区域
- 🟢 **下乐队 (Lower Cloud)** - 支撑位和超卖区域
- ➖ **中线 (Midband)** - 动态均衡位
- 🎨 **渐变层 (Gradient Layers)** - 波动率强度可视化
  {% endcolumn %}

{% column %}

#### 为什么有效

云形乐队实时适应市场波动率，提供关键感知的支撑和阻力位，这是静态指标无法做到的。多层渐变一目了然地显示强度水平。
{% endcolumn %}
{% endcolumns %}

{% hint style="info" %}
**交易优势：** 自适应特性意味着你总是在使用相关的位置 - 而非忽视当前市场条件的固定线。
{% endhint %}

### 📊 理解云形结构

#### 云层

云形由多个乐队组成，创建渐变效果：

| 层级                       | 位置       | 含义          | 交易用途   |
| -------------------------- | ---------- | ------------- | ---------- |
| **外层乐队 (Outer Band)**  | 远离价格   | 极端超买/超卖 | 反转区域   |
| **中间层 (Middle Layers)** | 渐变区域   | 强度水平递增  | 部分止盈   |
| **内层乐队 (Inner Band)**  | 最靠近价格 | 初始支撑/阻力 | 首个目标   |
| **中线 (Midband)**         | 中心线     | 动态均衡      | 偏差过滤器 |

#### 动态中线颜色编码

{% column width="50%" %}
**价格位置指示：**

- 🟢 **绿色** - 价格在上方（看涨）
- 🔴 **红色** - 价格在下方（看跌）
- ⚪ **中立** - 价格在均衡
  {% endcolumn %}

{% column %}
**交易应用：**

- 绿色 = 寻找做多机会
- 红色 = 寻找做空机会
- 中立 = 等待方向确认
  {% endcolumn %}

#### 周期模式指南

{% tabs %}
{% tab title="🤖 自动模式 (推荐)" %}
**基于时间框架 (Timeframe) 的自动调整：**

系统智能计算最优周期：

- **1-5 分钟：** 超快响应设置
- **15-60 分钟：** 平衡的日内交易 (Intraday)
- **4 小时-日线：** 平滑的波段交易 (Swing Trading)
- **周线及以上：** 仅主要趋势

**优势：**

- 无需手动调整
- 每个时间框架都已优化
- 减少虚假信号

**最适合：** 95% 的交易者
{% endtab %}

| **60-80**   | 平衡敏感度         | 日内交易 (Day Trading)      |
| **100-150** | 平滑，虚假突破少   | 波段交易 (Swing Trading)    |
| **200+**    | 仅主要趋势         | 持仓交易 (Position Trading) |

**何时使用：**

- 特定策略要求
- 回测一致性
- 个人偏好

**最适合：** 有特定需求的高级用户

### 🎯 交易信号

{% stepper %}
{% step %}

#### 云形突破 (Cloud Breakouts)

**看涨突破：**

- 价格收盘在上乐队上方
- 云形向上倾斜
- 成交量确认最佳

**看跌突破：**

- 价格收盘在下乐队下方
- 云形向下倾斜
- 强势动量信号
  {% endstep %}

{% step %}

#### 云形挤压 (Cloud Squeeze)

**特点：**

- 云形乐队收敛（变窄）
- 低波动率期间
- 能量为大幅波动积蓄

**交易方法：**

- 标记挤压区域
- 等待扩张方向
- 以紧密的止损进行突破交易
  {% endstep %}

{% step %}

#### 中线穿越 (Midband Crosses)

**信号：**

- 穿越上方 = 看涨偏差转变
- 穿越下方 = 看跌偏差转变
- 多次触及 = 强支撑位

**确认方式：**

- 检查云形斜率
- 验证多时间框架 (MTF) 对齐
- 查找成交量
  {% endstep %}
  {% endstepper %}

### 📈 交易策略

#### 策略 1：云边反弹

**配置：**

1. 价格在趋势市场中接近云形边缘
2. 寻找拒绝 K 线 (Rejection Candles)
3. 以紧密止损进行反弹入场
4. 目标为对面云形边缘或中线

**最适合：** 具有明确云形斜率的趋势市场

#### 策略 2：突破交易

**配置：**

1. 识别云形挤压（乐队变窄）
2. 等待成交量支持的决定性突破
3. 在云形边缘回测时进场
4. 止损在云形下方/上方
5. 目标 = 2 倍云形宽度

**最适合：** 范围突破和动量交易

#### 策略 3：均值回归 (Mean Reversion)

**配置：**

1. 价格在外层云形极端
2. 寻找反转形态
3. 进场目标为中线
4. 止损在云形极端之外

**最适合：** 超高/超低市场

### 🎨 视觉解读

#### 云形颜色和不透明度

{% columns %}
{% column width="50%" %}
**颜色强度：**

- **深色/不透明** = 高波动率
- **浅色/透明** = 低波动率
- **红色渐变** = 阻力区域
- **绿色渐变** = 支撑区域
  {% endcolumn %}

{% column %}
**交易含义：**

- 深色 = 使用更宽的止损
- 浅色 = 收紧风险
- 红色 = 卖出区域
- 绿色 = 买入区域
  {% endcolumn %}
  {% endcolumns %}

#### 云形斜率分析

| 斜率         | 视觉      | 含义       | 交易偏差 |
| ------------ | --------- | ---------- | -------- |
| **陡峭向上** | 📈 45°+   | 强上升趋势 | 仅做多   |
| **温和向上** | ↗️ 15-45° | 弱上升趋势 | 偏好做多 |
| **平坦**     | ➡️ 0-15°  | 盘整市场   | 双向交易 |
| **温和向下** | ↘️ 15-45° | 弱下降趋势 | 偏好做空 |
| **陡峭向下** | 📉 45°+   | 强下降趋势 | 仅做空   |

### ⚠️ 常见错误

{% hint style="danger" %}
**关键错误需避免：**

1. **与云形斜率交易相反** - 尊重趋势方向
2. **忽视挤压配置** - 这些提供最佳风险/收益比
3. **错误的周期设置** - 从自动模式开始
4. **在云形中部进场** - 在边缘进场以获得最佳入场
5. **不根据宽度调整止损** - 宽云形需要宽止损
   {% endhint %}

### 🛠️ 故障排除

<details>

<summary><strong>云形不显示</strong></summary>

- 检查 "显示云形乐队" 已启用
- 确保加载了足够的价格历史数据
- 尝试在自动/手动模式之间切换
- 验证图表是否有足够的数据点
- 某些工具可能需要手动调整周期

</details>

<details>

<summary><strong>云形太宽/太窄</strong></summary>

- **自动模式：** 自动调整 - 如果仍然不对，尝试手动模式
- **手动模式调整：**
  - 太宽 → 降低周期（尝试 -20）
  - 太窄 → 提高周期（尝试 +20）
- 考虑你的时间框架（低 TF = 通常较低周期）

</details>

<details>

<summary><strong>中线不可见</strong></summary>

- 在设置中启用 "显示云形中线"
- 可能被价格 K 线隐藏
- 检查是否与背景色相同
- 尝试放大/缩小

</details>

<details>

<summary><strong>云形显示不平稳</strong></summary>

- 在非常低的时间框架 (1-5 分钟) 上正常
- 解决方案：
  - 在手动模式中提高周期
  - 使用更高时间框架 (15 分钟+)
  - 切换到自动模式
  - 如果可用，启用平滑处理

</details>
