#### 🎯 快速概览

**功能说明**

- 显示机构订单流区域
- 实时监控 8 个时间框架
- 识别市场结构变化
- 显示自适应波动率带状线

**适合**

- 多时间框架交易者
- 机构订单流分析
- 趋势跟踪策略
- 支撑/阻力（Support/Resistance）识别

---

#### 📚 基本设置

**从这里开始**

1. [**订单块（Order Blocks）**](https://infinity.aloen.to/dashboard/infinity-dashboard/order-blocks)- 设置为"仅显示最近"（Show Nearest Only）以获得清晰视图
2. [**多时间框架仪表盘（MTF Dashboard）**](https://infinity.aloen.to/dashboard/infinity-dashboard/mtf-dashboard)- 根据您的风格选择时间框架
3. [**云带（Cloud Bands）**](https://infinity.aloen.to/dashboard/infinity-dashboard/cloud-bands) - 在大多数市场中使用自动模式
4. [**警报（Alerts）**](https://infinity.aloen.to/dashboard/infinity-dashboard/dashboard-alerts) - 启用多时间框架对齐警报

**高级功能**

- **市场结构（Market Structure）** - BOS/CHoCH 检测
- **成交量分析（Volume Analysis）** - 区域内的买卖压力
- **主题自定义（Theme Customization）** - 匹配您的交易环境
